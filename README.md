## Code setup

1. Creating a `.env` environment file and add the following to it
    ```
    ROBOFLOW_KEY=<your_key>
    ```
2. Spin up the server
    ```
    uvicorn main:app --host 0.0.0.0 --port 8080 --reload
    ```
